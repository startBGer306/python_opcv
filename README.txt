#此代码用于在PC端进行调试，请在实际使用时更改摄像头的序号以及对应的串口号 ！！！！
请记住端口的波特率为115200 ！！！！
程序的整体框架
1：初始化摄像头，并且是摄像头成为全局变量                                          init_camera()
2：由于红色阈值有两个因此每个检测函数都有两个，一个用于通用的蓝色和绿色一个用于红色    detect_color()
3：中间的函数结构为色块检测，红色专属色块检测                                       detect_color_red()
4：色环检测，红色色环专属检测                                                     detect_circle(),detect_circle_red()
5：全色块检测                                                                    Det_all()
6：空线程的函数                                                                  Donothing()
7：数据转化函数用于传输给32数据                                                   datatransfrom()
8：全色块检测中使用的重复部分函数                                                  det_for()
9：串口的管理类                                                                  class SerialListener()继承于线程类
10：主函数处理开启摄像机打开串口线程，初始化任务变量 main
11：转换线程路逻辑
12：结束逻辑
